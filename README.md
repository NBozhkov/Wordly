# Wordly
## The Ultimate Helper for Learning Words
### Wordly is an offline Windows compatible application made with C# on the .NET framework.
## Key Idea
### The aim of the program is to assist one's learning of a certain set of terms(words), supposedly in a foreign language, although it can also be used in different cases where the goal is simply memorization.
### It achieves this through different carefully selected and research driven methods including: simple revising, thorough concentration over each item, a simulation of a final test which grades to what extent the set is memorized, and more. Each method offers a variety of tools which ease, speed up the process and make it more enjoyable and reliable compared to studying from paper source.
